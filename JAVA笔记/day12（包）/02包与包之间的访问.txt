一、
当一个类有所属包的时候，要调用这个类，就要明确其包名，也就是要如下调用：
	包名.类名

当一个类又所属包时，但是这个包又不在当前目录下，要怎样调用这个包？
	那么，可以通过设置类文件路径，比如有过一个包：在E:\ljc，那么可以通过如下语句设置类文件路径：
	set classpath=e:\ljc
	此时，把包和类当成一个整体，即一个完整的类，应该依赖于包存在的

public :
包与包之间的类进行访问，被访问的包中的类必须是public修饰的，被访问的包中的类中的方法必须是public修饰的

protected:
protected修饰的方法在不同包中引用时，只有继承了该类的子类对象才可以使用（访问），就算是建立它本身的对
象也是无法访问的，只有先用一个子类来继承该类，然后才能通过该子类建立对象来访问
也就是说，在不同包中类之间的访问情况下，protected修饰的方法是为其子类准备的，不是自雷不能访问
