一、
TreeMap是Map的一个子类，该类的内部结构是二叉树，一键值对形式存储

二、
TreeMap集合中的元素都是有序的，该有序是通过键值对中的 键 的比较方法来实现的，可以说元素的有序就是键的有序
因此想要往TreeMap集合添加键值对，键一定要实现Comparable接口并覆盖compareTo方法

三、
实现元素的排序，还可以通过在集合创建时传递给它比较器的方式来实现
该比较器就是一个实现了Comparator接口的类的对象，该类中必须覆盖Compare方法，示例如下：
public int compare(Object o1,Object o2){
	return 0;
}
此方法中参数1往往代表的是需要添加的元素，参数2往往代表的是已有的元素，参数1大就返回正数，小返回负数，等于返回0
而且，该方法比较的都是键！
