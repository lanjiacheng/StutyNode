一、异常处理的捕捉形式：
这是可以对异常进行针对性处理的方式

具体格式：
try
{
	//需要被检测异常的代码
}
catch(异常类 变量)//该变量用于接收发生的异常对象
{
	//处理异常的代码
}
finally
{
	//一定会被执行的代码，并且该部分的代码是在catch之后执行的
}

二、处理方式：
当异常被捕捉时，可以在catch语句中进行如下处理：
1.getMessage():返回异常信息，就是抛出异常时传递给异常类构造方法的信息
2.e.toString():返回异常对象的字符串信息，直接打印对象名其实就是通过这个方法来实现的
3.e.printStackTrace();jvm默认的异常处理机制就是调用异常对象的这个方法

三、当出现多个catch语句时，一般按顺序执行，并且只执行一个catch，但是当出现父类的catch
	时，父类的一定要排在后面，否则编译失败
