循环语句的代表：while、do while、for

一、while
格式：
while(条件表达式)
{
执行语句;
}

二、do while
格式：
do
{
执行语句;
}while(条件表达式);
说明：无论条件是否满足，循环都执行一次

三、for
格式：
for（初始化表达式;循环条件表达式;循环后的操作表达式)
{
执行语句;
}
说明：for语句可以在初始化表达式语句中定义并初始化变量，该变量在for循环结束后被释
放。for语句三表达式都可以不写，但是中间的条件表达式不写，默认为true

一个表达式中有多个语句时，语句之间用逗号,隔开

最简单无限循环表达式：
while(true)
{

}
for(;;)
{

}
