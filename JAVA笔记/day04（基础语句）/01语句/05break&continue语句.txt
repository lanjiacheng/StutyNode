一、break语句：
几点说明：
1.break作用：跳出当前循环
2.作用范围：要么是switch语句要么是循环语句
3.注意：当break单独存在时，下面不要定义其他语句，因为执行不到，编译报错
4.当多循环嵌套时，break语句跳出当前所在循环
5.如果在多循环嵌套语句中，break想要跳出指定循环，可以通过标号来完成，例如：
	for1:for(;;)//此处给外循环标号为：for1
	{
		for2:for(;;)//此处给内循环标号为：for2
		{
			break for1;//此处跳出外循环
		}
	}
二、continue语句：
几点说明：
1.continue作用：结束本次循环，继续下次循环
2.作用范围：只能在循环语句中使用
3.注意：当continue单独存在时，下面不要定义其他语句，因为执行不到，编译报错
4.当多循环嵌套时，continue语句继续的是当下的循环
5.当多循环嵌套时，continue要作用于指定循环，可以通过标号来实现，具体语法，参考break语句
